# Divyanshu Raj — Portfolio

Personal portfolio website for Divyanshu Raj, Backend Engineer.

Built with **React + Vite**, **Tailwind CSS**, **Framer Motion**, and **React Router DOM**.

## 🌟 Featured Projects
- **QueryVault**: Multi-Modal RAG AI Chatbot (Llama 3 70B, Pinecone Serverless, FastAPI, Supabase AES Encryption).
- **Task Management System**: Enterprise Full-Stack Application (Next.js + Express.js + PostgreSQL).
- **Fitness Tracker**: Full-Stack Spring Boot microservice.


## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
# → Opens at http://localhost:5173

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📁 Adding Your Assets

Before deploying, add the following files to the `public/` directory:

| File | Description |
|------|-------------|
| `public/images/profile.jpg` | Your profile photo (shown in Hero & About) |
| `public/images/chatbot.png` | QueryVault AI Chatbot screenshot |
| `public/images/project-taskmanager.png` | Task Manager screenshot |
| `public/images/project-fitnesstracker.png` | Fitness Tracker screenshot |
| `public/resume.pdf` | Your resume PDF (linked from Hero) |

Styled placeholders are shown until you add the images.

## 🌐 Deploy to Vercel (Free)

```bash
# 1. Initialize git & push to GitHub
git init
git add .
git commit -m "Initial portfolio"
git remote add origin https://github.com/spidyraj/divyanshu-portfolio.git
git push -u origin main
```

2. Go to [vercel.com](https://vercel.com) → Sign in with GitHub
3. Click **Add New Project** → Import your repo
4. Framework: **Vite** (auto-detected)
5. Click **Deploy** ✅

**Live URL**: `https://portfoliodivyanshu.vercel.app/`

> **Custom domain**: Vercel → Project Settings → Domains
